# UX Copy

> Microcopy en **français**, ton du produit (clair, rassurant, professionnel). À écrire/réviser avec la skill **`ux-copy`**.

À couvrir :
- **CTA** (boutons), titres d'écrans, libellés de champs.
- **États vides** (pas de RDV, pas de document, pas de message).
- **Messages d'erreur** (paiement échoué, créneau indisponible, hors délai d'annulation).
- **Confirmations** (RDV pris, devis signé, acompte payé).
- **Notifications** (textes courts, **sans donnée de santé** dans le push — cf. `../../docs/04` §8).

Garde-fou : **aucune promesse clinique** ni conseil médical automatisé dans la copy (cf. `../../docs/03` §2, `../../docs/07` §8).
