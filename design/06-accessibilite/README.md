# Accessibilité

> Audit **WCAG 2.1 AA / RGAA** avec la skill **`accessibility-review`**. App santé grand public → l'accessibilité est un enjeu réel (seniors, handicaps), pas une option.

À vérifier :
- **Contrastes** (texte, composants, états).
- **Tailles de cible** tactiles (mobile patient).
- **Navigation clavier** + focus visible (back-office desktop).
- **Lecteurs d'écran** : labels, rôles, ordre de lecture.
- **Lisibilité** : tailles de police, hiérarchie, langue déclarée (fr).

À faire avant le `07-handoff/`.
